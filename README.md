# Library Management System (Python)

## Overview
A simple library management system built with Python to manage books and members, and support basic library operations.

## Features
- Add / remove / search books
- Register members
- Borrow and return books
- View available books and member records

## Key Concepts Demonstrated
- File handling (read/write)
- Data parsing and validation
- Role-based access control
- Business rule implementation (loan limits, overdue checks)
- Modular function design
